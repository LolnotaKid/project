HOW MANY FEATURES ARE AVAILABLE?

  • Run VFX - Allows you to run the VFX's path to see how it looks. There's also a "Generate" button that converts the VFX path into a script so you can use it.
  • VFX Stealer - Steals VFXs from the current game.
